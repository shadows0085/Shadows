SHADOW — Video Upload Folder
══════════════════════════════

Drop your MP4 files here. Then update data/videos.json to point to them.

Naming convention used by default:
  edit-01.mp4  →  Ephemeral Moments  (Commercial)
  edit-02.mp4  →  Amber Hours        (Motion)
  edit-03.mp4  →  Silent Waters      (Film)
  edit-04.mp4  →  Velocity           (VFX)
  edit-05.mp4  →  Forest Pulse       (Film)
  edit-06.mp4  →  Neon Reverie       (Motion)

To add a new project:
  1. Drop your video here (e.g. edit-07.mp4)
  2. Open data/videos.json
  3. Add a new entry to the "portfolio" array
  4. Refresh the website — it appears automatically

See data/videos.json → "_instructions" for full field reference.
